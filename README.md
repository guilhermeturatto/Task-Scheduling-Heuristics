## To run the program:

- Run "SchedulingHeuristics.exe" to open the interface (may take a few seconds and open in the background);

or

- Open source code in python and run with "python SchedulingHeuristics.py" in the terminal;


<br />

## Comments:

- The data sheet must be in the same folder as the program;

- The data sheet must be closed when clicking "Start";

- The data is read and the results are recorded automatically when clicking "Start";

- The spreadsheet can be opened and the results will be updated after the "Calculations Completed";

- Changes to the data can be made while the program is running, but you must save the changes and close the worksheet before clicking "Start";
